=== Wordpress Disable All Comments ===
Contributors: lowest
Tags: wp comments, wp remove comments, wp disable comments, hide comments, disable wp comments, lightweight
Requires at least: 1.2.0
Stable tag: 1.0
Tested up to: 5.0.2

== Description ==
Disable all wordpress comments including past/old posts as well as future posts.
If posts had comments they will simply not display instead visitors will see comments disabled in the comments section
